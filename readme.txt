== Description ==

This plugin gives a nice dutch-supporter (soccer) supporter attachment (orange) to any wordpress-website/blog.


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `wk-mood.php` and the rest (including jquery.js) to the `/wp-content/plugins/wk-mood/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('wk-mood_hook'); ?>` in your templates

== Frequently Asked Questions ==

= Is the plugin fully selfexplaining? =

yes, it is just a regular plugin, no options.

= Is the plugin only for the dutch soccer? =

yes, but anyone can use it!.

= Is the plugin content static or dynamic? =

The content is dynamic and up to date. During the word championship a variety of little movies will be shown in the plugin!.
